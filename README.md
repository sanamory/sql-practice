# 📊 SQL Practice & Analytics Portfolio

Welcome to my SQL practice repository! This is where I keep my SQL code, database designs, and data analysis projects to show my growth and skills.

---

## 🚀 Project Overview
The main goal of this project is to write clean, organized, and efficient SQL queries to solve real-world problems. I focus on building reliable databases and transforming raw numbers into useful insights.

* **Database Tool:** Microsoft SQL Server (T-SQL)
* **What I Practice:** Creating tables, sorting data, joining tables, and building financial/business reports.

---

## 📂 Repository Structure

I organize my work into simple folders so it is easy to browse:

* `📁 lesson1/` — The basics: Creating new tables, setting up auto-incrementing IDs (identity columns), setting primary keys, and importing data from Excel and CSV files.
* *`📁 business-analysis/` (Coming Soon)* — Advanced practice: Combining tables (JOINs), grouping data, and writing complex queries for business and sales reports.

---

## 💡 Key SQL Skills I am Building

* **Smart Database Design:** Organizing tables correctly so the database runs fast and without errors.
* **Data Accuracy:** Making sure data types match, handling missing information (NULL values), and moving data into SQL from Excel smoothly.
* **Problem Solving:** Writing queries that answer business questions (like finding top sales, calculating monthly growth, etc.).
